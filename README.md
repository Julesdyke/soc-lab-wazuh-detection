# soc-lab-wazuh-detection
Built a functional Security Operations Center (SOC) using Wazuh SIEM to detect, analyze, and respond to simulated cyberattacks including brute-force attempts, account manipulation, and vulnerability exploitation.

What is Security Operations Center (SOC)? 
A Security Operations Center (SOC) is a centralized system that collects, analyzes, and monitors security events across devices to detect and respond to threats in real time. Many organizations rely on SOCs to identify and respond to cyberattacks as they happen.
